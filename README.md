## women_health Monorepo

This is a Node.js, TypeScript, and React monorepo for the women health web application.

### Structure

- `apps/`
  - `web/`: React + TypeScript web application
- `packages/`
  - `api-client/`: Placeholder for shared API client
  - `shared-i18/`: Placeholder for shared i18n utilities
  - `shared-types/`: Placeholder for shared TypeScript types
  - `ui-components/`: Placeholder for shared UI components
  - `utils/`: Placeholder for shared utilities

### Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the web app:

```bash
cd apps/web
npm run dev
```
